Mozilla Firefox Setup for CMS400

Firefox CMS400 Plugin Setup Instructions

There is a manual step to configure the Mozilla Firefox browser to run with the CMS400 product. A plug-in file must be downloaded and installed into the Firefox plugins directory.

Select the link below to begin the download process:

Download Firefox Plugin (np72esk32.dll)

The browser will prompt you with options on how to process the file. Choose the "Save to Disk" option to save the file to your local system.

If you are using the Firefox browser to download this file it will place the plugin file automatically into your configured download directory. You will need to manually copy this file into your Firefox plugins directory.

The default plugins directory for Firefox is:

C:\Program Files\Mozilla Firefox\plugins

After the plugin file is placed into the Firefox plugins directory then all CMS400 content is editable from within the Mozilla Firefox browser.
